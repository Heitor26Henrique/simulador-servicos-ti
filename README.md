# 💻 Simulador de Serviços de TI

## 📌 Descrição
Este projeto simula a contratação de serviços de TI, permitindo ao usuário selecionar diferentes serviços e visualizar o valor total em tempo real.

## ⚙️ Funcionalidades
- Seleção de serviços
- Soma automática dos valores
- Lista de serviços escolhidos
- Botão para limpar seleção

## 💰 Serviços disponíveis
- Landing Page
- E-commerce
- Integração de API
- Manutenção

## 🛠️ Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## 📊 Base de preços
Os valores foram definidos com base em pesquisas em plataformas como 99Freelas, Workana, Contabilizei e Glassdoor.

## 🚀 Como executar
1. Baixe os arquivos
2. Abra o arquivo index.html no navegador
